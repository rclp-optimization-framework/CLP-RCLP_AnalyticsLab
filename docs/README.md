# Documentation Index

This folder organizes the project's documentation by topic:

- `architecture/` — System architecture and design notes
- `api/` — API reference and contract notes (transactional and statistics)
- `guides/` — Getting started, testing, deployment, and quick reference
- `examples/` — Sample payloads and battery examples
- `research/` — Research artifacts and model sources (MathModel.tex, features)

Use these documents when developing, testing, or releasing the project.
